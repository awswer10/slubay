Use Case

Title: Create a post
Actor: Any user
Prerequisite: User has an account
Action:
  • Visit post creation page
  • Submit form
  • If Successful: Post is confirmed as created
  • If Failed: User recieves an error message
    		•
    Effect: Post is created by the system
    Next: View the newly created post's page
