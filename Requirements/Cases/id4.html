Use Case

Title: Edit a post
Actor: Any user
Prerequisite: User has created a post.
Action:
  • View all posts owned by user
  • Select post to edit
  • Open post editing form
  • Submit edited form and confirm submission
    		•
    Effect: System changes post to match edits made by user
    Next: View edited post
